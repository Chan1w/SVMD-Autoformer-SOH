## 📖 Paper Information

**Title:**  
*A Lightweight Multiscale Signal Learning Framework for Predicting Battery Degradation Trajectory*

**Journal:** IEEE Sensors Journal  
**Year:** 2025  

This repository provides the official PyTorch implementation of the proposed framework integrating:

- 🔹 Successive Variational Mode Decomposition (SVMD)  
- 🔹 Period-Sensitive Auto-Correlation Module (ACM)  
- 🔹 Lightweight Spatiotemporal MLP  
- 🔹 Bayesian Hyperparameter Optimization  

---

## 🧠 Motivation

Battery degradation signals exhibit:

- Nonlinearity  
- Nonstationarity  
- Multiscale temporal coupling  
- Periodic regeneration effects  

Conventional RNN/Transformer-based models:

- ❌ High computational complexity  
- ❌ Over-parameterization  
- ❌ Poor interpretability of multiscale components  

SVMD-FAC-net addresses these challenges via:

- ✔ Multiscale signal decoupling  
- ✔ Frequency-aware modeling  
- ✔ Lightweight parallel networks  
- ✔ Efficient inference
